<h2><a id="user-content-lists" class="anchor" aria-hidden="true" href="#lists"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Lists</h2>
<p>There are four collection data types in python :</p>
<ul>
<li>List: is a collection which is ordered and changeable(modifiable). Allows duplicate members.</li>
<li>Tuple: is a collection which is ordered and unchangeable or unmodifiable(immutable). Allows duplicate members.</li>
<li>Set: is a collection which is unordered, unindexed and unmodifiable, but you can add new items. No duplicate members.</li>
<li>Dictionary: is a collection which is unordered, changeable(modifiable) and indexed. No duplicate members.</li>
</ul>
<p>A list is collection of different data types which is ordered and modifiable(mutable). A list can be empty or it may have different data type items or items</p>
<h3><a id="user-content-how-to-create-a-list" class="anchor" aria-hidden="true" href="#how-to-create-a-list"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>How to Create a List</h3>
